---
title: Get Dataset Version Content
openapi: get /datasets/{dataset_id}/versions/{version_index}/content
---

This endpoint retrieves the content of a specific version of a dataset. This allows you to view historical data and compare changes between versions.

## Response

| Field                 | Type             | Description                                                           |
| --------------------- | ---------------- | --------------------------------------------------------------------- |
| `starting_token`      | integer          | The starting token used for this request                              |
| `limit`               | integer          | The limit used for this request                                       |
| `paginated`           | boolean          | Whether the results are paginated                                     |
| `next_starting_token` | integer or null  | Token to use for the next page of results, or null if no more results |
| `column_names`        | array of strings | Names of columns in the dataset version                               |
| `rows`                | array of objects | List of dataset row objects in this version                           |

### Row Object

| Field      | Type           | Description                                                                 |
| ---------- | -------------- | --------------------------------------------------------------------------- |
| `row_id`   | string (UUID)  | Unique identifier for the row                                               |
| `index`    | integer        | Index of the row in the dataset                                             |
| `values`   | array          | Values for each column in the row (can be string, number, integer, or null) |
| `metadata` | object or null | Additional metadata associated with the row                                 |

## Notes

- This endpoint is useful for comparing different versions of a dataset
- Use the Query Dataset Versions endpoint to find available version indexes
- The column_names array provides the names of columns in the same order as the values in each row
- Use pagination parameters to efficiently retrieve large datasets
- If the dataset or version doesn't exist, or the user doesn't have permission to access it, a 404 error will be returned
