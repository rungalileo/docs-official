---
title: Get Dataset
openapi: get /datasets/{dataset_id}
---

This endpoint retrieves detailed information about a specific dataset by its unique identifier. It provides comprehensive metadata about the dataset, including its structure and permissions.

## Response

| Field           | Type               | Description                             |
| --------------- | ------------------ | --------------------------------------- |
| `id`            | string (UUID)      | Unique identifier for the dataset       |
| `name`          | string             | Name of the dataset                     |
| `created_at`    | string (date-time) | Creation timestamp                      |
| `updated_at`    | string (date-time) | Last update timestamp                   |
| `project_count` | integer            | Number of projects using this dataset   |
| `num_rows`      | integer            | Number of rows in the dataset           |
| `column_names`  | array of strings   | Names of columns in the dataset         |
| `permissions`   | array of objects   | Permissions associated with the dataset |

## Notes

- This endpoint returns detailed metadata about the dataset but not its actual content
- To retrieve the dataset content, use the Get Dataset Content endpoint
- If the dataset doesn't exist or the user doesn't have permission to access it, a 404 error will be returned
