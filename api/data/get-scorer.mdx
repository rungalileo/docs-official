---
title: Get Scorer
openapi: get /scorers/{scorer_id}
---

This endpoint retrieves detailed information about a specific scorer in your Galileo account. Scorers are evaluation tools that help you assess the quality and performance of your AI model outputs. This endpoint provides comprehensive details about the scorer's configuration, versions, and metadata.

## Response

The response contains detailed information about the requested scorer.

## Error Responses

| Status Code | Description |
| ----------- | ----------- |
| `404` | Scorer not found |
| `403` | Insufficient permissions to access the scorer |

## Notes

- Use the `include_versions` parameter to retrieve the full version history of the scorer
- The `latest_version` field provides a quick reference to the most recent version
- The `defaults` object contains the default configuration settings for the scorer
- The `scorer_type` field indicates whether this is an LLM-based, code-based, or preset scorer
- For detailed information about a specific version, use the Get Scorer Version or Latest endpoint
- For code-based scorers, use the Get Scorer Version Code endpoint to retrieve the actual code implementation
