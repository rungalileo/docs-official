---
title: List Scorer Tags
openapi: get /scorers/tags
---

This endpoint retrieves a list of all tags used across scorers in your Galileo account. Tags are labels that help categorize and organize scorers, making them easier to find and manage. This endpoint is particularly useful for discovering available tags for filtering scorers or maintaining consistent tagging conventions.

## Query Parameters

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `prefix` | string | Optional. Filter tags that start with the specified prefix |
| `limit` | integer | Optional. Maximum number of tags to return (default: 100, max: 500) |

## Response

The response contains a list of unique tags used across all scorers.

## Error Responses

| Status Code | Description |
| ----------- | ----------- |
| `403` | Insufficient permissions to list tags |
| `400` | Invalid query parameters |

## Security

This endpoint requires authentication. You can authenticate using:

- An API key in the `Galileo-API-Key` header
- OAuth2 password flow
- HTTP Basic authentication

## Notes

- The `count` field indicates how many scorers are using each tag
- Use the `prefix` parameter to search for tags that match a specific naming convention
- Tags are case-sensitive, so "Production" and "production" would be considered different tags
- This endpoint is useful for discovering available tags when filtering scorers with the List Scorers with Filters endpoint
- Tags can help establish a consistent taxonomy for organizing and categorizing scorers
- Consider using a standardized tagging convention to make scorers more discoverable and manageable
