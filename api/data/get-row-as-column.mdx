---
title: Get Row as Column
openapi: get /projects/{project_id}/runs/{run_id}/prompts/rows/{row_id}/columnar
---

This endpoint retrieves detailed information about a specific row in a columnar format. It provides a structured view of the prompt, completion, and associated metadata, making it easier to analyze and visualize the data. The columnar representation is particularly useful for integrating with data analysis tools and dashboards.

## Path Parameters

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `project_id` | string (UUID) | The unique identifier of the project |
| `run_id` | string (UUID) | The unique identifier of the experiment run |
| `row_id` | string (UUID) | The unique identifier of the row to retrieve |

## Query Parameters

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `include_metrics` | boolean | Optional. Whether to include evaluation metrics (default: false) |
| `include_metadata` | boolean | Optional. Whether to include additional metadata (default: false) |

## Response

The response contains detailed information about the specified row in a columnar format.

## Error Responses

| Status Code | Description |
| ----------- | ----------- |
| `404` | Project, run, or row not found |
| `403` | Insufficient permissions to access the data |
| `400` | Invalid query parameters |

## Security

This endpoint requires authentication. You can authenticate using:

- An API key in the `Galileo-API-Key` header
- OAuth2 password flow
- HTTP Basic authentication

## Notes

- The columnar format provides a structured representation of the row data, with each field represented as a named column
- Each column includes a name, value, and type, making it easier to understand and process the data
- The `include_metrics` parameter adds evaluation metrics to the response, which can be useful for performance analysis
- The `include_metadata` parameter adds additional context about the row, such as template information and user details
- This endpoint is particularly useful for detailed analysis of individual prompts and completions
- The columnar format makes it easy to integrate with data visualization tools and dashboards
- For analyzing multiple rows, consider using the Get Rows as Columns with Filters endpoint
