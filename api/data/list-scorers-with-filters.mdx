---
title: List Scorers with Filters
openapi: post /scorers/list
---

This endpoint retrieves a list of scorers in your Galileo account with powerful filtering capabilities. It allows you to search, sort, and filter scorers based on various criteria, making it easy to find specific scorers or groups of scorers that match your requirements.

### Example Request

```json
{
  "filters": [
    {
      "field": "name",
      "operator": "contains",
      "value": "customer"
    },
    {
      "field": "created_at",
      "operator": "gte",
      "value": "2023-01-01T00:00:00Z"
    }
  ],
  "sort_by": {
    "field": "created_at",
    "direction": "desc"
  },
  "page": 1,
  "page_size": 20,
  "include_versions": true,
  "tags": ["production", "customer-support"],
  "scorer_types": ["llm", "preset"]
}
```

## Response

The response contains a list of scorers that match the specified filters.

## Error Responses

| Status Code | Description                              |
| ----------- | ---------------------------------------- |
| `403`       | Insufficient permissions to list scorers |
| `400`       | Invalid request body or parameters       |

## Notes

- Use the `filters` parameter to narrow down the list of scorers based on specific criteria
- The `tags` parameter provides a convenient way to filter scorers by their associated tags
- The `scorer_types` parameter allows you to focus on specific types of scorers
- Setting `include_versions` to true provides version history for each scorer, which can be useful for tracking changes
- For large numbers of scorers, use pagination to manage response size and loading times
- The response structure is similar to the Get Scorer endpoint, but with multiple scorers and pagination information
