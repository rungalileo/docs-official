---
title: Delete Scorer
openapi: delete /scorers/{scorer_id}
---

This endpoint allows you to permanently delete a scorer from your Galileo account. Deleting a scorer removes all associated versions and configurations. This action cannot be undone, so use it with caution.

## Notes

- This operation permanently deletes the scorer and all its versions
- Any evaluations that have already been performed using this scorer will remain in your logs
- If you want to keep the scorer for reference but prevent its use, consider using tags or other metadata to mark it as deprecated instead of deleting it
- Before deletion, ensure that no active workflows or automations depend on this scorer
